# 杰仕丝带公司官方网站

这是一个现代化的企业网站，包含公司介绍、零售产品和订制产品三大板块。

## 🚀 快速开始（超简单步骤）

### 第一步：安装 Node.js
1. 打开浏览器，访问：https://nodejs.org/
2. 下载并安装最新版本的 Node.js（选择 LTS 版本）
3. 安装完成后，重启电脑

### 第二步：打开项目文件夹
1. 在文件管理器中找到这个项目文件夹：`web jazser`
2. 在文件夹空白处，按住 `Shift` 键，然后右键点击
3. 选择"在此处打开 PowerShell 窗口"（或"在此处打开命令窗口"）

### 第三步：安装依赖
在打开的窗口中，复制粘贴下面这行命令，然后按回车：

```bash
npm install
```

等待安装完成（可能需要几分钟，看到 "added XXX packages" 就表示完成了）

### 第四步：启动网站
安装完成后，复制粘贴下面这行命令，然后按回车：

```bash
npm run dev
```

### 第五步：查看网站
看到类似这样的提示：
```
  VITE v5.x.x  ready in xxx ms

  ➜  Local:   http://localhost:3000/
```

就表示网站已经启动成功了！

**现在打开浏览器，访问：http://localhost:3000**

## 🎨 网站功能

### 1. 公司介绍板块
- 企业品牌展示
- 发展历程时间轴
- 核心优势介绍

### 2. 零售产品板块
- 产品展示和筛选
- 产品详情页面
- 购物车功能（待完善）

### 3. 订制产品板块
- 定制需求表单
- 流程展示

## 🛠️ 常用命令

- `npm run dev` - 启动开发服务器（用于查看网站）
- `npm run build` - 打包网站（用于上线部署）
- `npm run preview` - 预览打包后的网站

## 📝 下一步需要你做的事情

1. **准备公司 Logo**
   - 将公司 Logo 图片放到 `public` 文件夹
   - 文件名改为 `logo.svg` 或 `logo.png`

2. **准备产品图片**
   - 收集所有产品的图片
   - 我会帮你添加到产品展示中

3. **填写公司信息**
   - 联系方式（电话、邮箱、地址）
   - 公司简介文字
   - 发展历程的具体内容

4. **准备产品数据**
   - 产品名称、价格、描述
   - 产品分类信息
